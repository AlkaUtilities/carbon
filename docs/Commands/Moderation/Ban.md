- Description: Bans a user from the server

Usage: `/ban <user*: User> <reason: String>`

- Parameters:
	- **user**:
		- **Type**: [User](../../Types/User.md)
		- **Description**: User to ban (required)
		- **Required**: True
	- **reason**:
		- **Type**: [String](../../Types/String.md)
		- **Description**: Reason for banning the user
		- **Required**: No
