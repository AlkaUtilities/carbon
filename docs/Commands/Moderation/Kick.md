- Description: Kicks a user from the server

- Usage: `/kick <user*: User> <reason: String>`

- Parameters:
	- **user**:
		- **Type**: [User](../../Types/User.md)
		- **Description**: User to kick (required)
		- **Required**: No
	- **reason**:
		- **Type**: [String](../../Types/String.md)
		- **Description**: Reason for kicking the user
		- **Required**: No
